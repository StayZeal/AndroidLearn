#### 可达性分析

GCRoot

1、虚拟机栈中引用的对象

2、静态变量引用的对象

3、常量引用的对象

4、本地方法栈（native 方法）引用的对象

#### 垃圾收集算法

标记-清除

标记-整理

复制算法：留出一般个空间进行复制。

分带收集算法：

- 新生代：留出一少部分内存用来进行复制，因为大部分对象都已经死亡。

- 老年代：标记-清除算法/标记整理算法
